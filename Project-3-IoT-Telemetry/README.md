# Project 3: Cloud-Connected Security Node (IoT Telemetry)

## Overview

This project demonstrates an IoT-based security monitoring system that collects sensor data and transmits it to a cloud dashboard using internet connectivity.

## Objective

Build a cloud-connected security node capable of:

* Monitoring physical events using sensors
* Sending telemetry data through Wi-Fi
* Using MQTT or HTTP protocols
* Displaying live data on a cloud dashboard

## Technologies Used

* ESP8266 / ESP32
* Ultrasonic Distance Sensor
* Wi-Fi Network
* MQTT / HTTP POST
* Adafruit IO / Blynk Dashboard

## Workflow

1. Sensor captures distance measurements.
2. ESP8266/ESP32 processes the readings.
3. Data is transmitted through Wi-Fi.
4. MQTT or HTTP sends telemetry to the cloud.
5. Dashboard displays live sensor data.
6. Alerts are generated when suspicious activity is detected.

## Outcome

A simulated IoT telemetry system that demonstrates cloud connectivity, real-time monitoring, and dashboard visualization.

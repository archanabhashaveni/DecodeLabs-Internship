# Cloud Dashboard Design

## Platform

Adafruit IO / Blynk Dashboard

## Dashboard Widgets

### Distance Monitor

Displays live distance readings from the ultrasonic sensor.

### Intruder Alert

Shows warning messages when the detected distance falls below a predefined threshold.

### Live Data Graph

Visualizes sensor readings over time.

## | Distance Reading | Intruder Alert Status |

## Alert Conditions

Distance < 30 cm

Status: ALERT - Possible Intruder Detected

Distance ≥ 30 cm

Status: Normal Monitoring

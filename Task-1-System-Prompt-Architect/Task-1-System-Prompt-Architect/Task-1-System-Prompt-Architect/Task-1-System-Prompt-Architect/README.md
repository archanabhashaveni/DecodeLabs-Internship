# 🧠 Task 1: System Prompt Architect

## 📌 Overview
This project demonstrates System Prompt Engineering and Persona Design by creating a Luxury Travel Consultant AI capable of handling premium customer interactions.

---

## 🎯 Objective
Design a structured system prompt that:
- Defines a strict AI persona
- Enforces business rules and constraints
- Maintains consistent tone
- Handles difficult customer interactions
- Uses few-shot prompting examples

---

## 🧩 Files Included

- system_prompt.md → AI persona and rules
- test_case.md → Customer scenario
- ai_response.md → Model output response
- README.md → Project documentation

---

## 🧪 Test Scenario
The AI was tested with a difficult customer request involving:
- Demand for 50% discount
- Request for visa guarantee
- Competitor comparison pressure

---

## 🤖 Outcome
The AI successfully:
- Maintained luxury consultant persona
- Rejected invalid discount request
- Clearly refused visa guarantee
- Avoided competitor references
- Provided premium alternative solutions

---

## 🚀 Skills Demonstrated
- System Prompt Engineering
- Persona Design
- Few-shot Prompting
- Constraint-based AI behavior design
- Conversation control under pressure

---

## 🏁 Conclusion
The system prompt effectively ensures consistent, professional, and premium travel consultation behavior without breaking character under difficult customer conditions.

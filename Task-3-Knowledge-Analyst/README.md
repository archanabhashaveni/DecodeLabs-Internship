# Task 3: The Knowledge Analyst (RAG Concepts)

## Overview

This project demonstrates Retrieval-Augmented Generation (RAG) concepts for document intelligence.

## Objective

Simulate a legal document analysis workflow that:

* Summarizes key clauses
* Answers questions with citations
* Extracts risks
* Identifies important dates
* Identifies stakeholders

## Files Included

* sample_document.md
* rag_prompt.md
* questions_and_answers.md
* summary_dashboard.md
* README.md

## Outcome

A simulated RAG workflow that minimizes hallucinations by requiring citation-based answers.

## Skills Demonstrated

- Retrieval-Augmented Generation (RAG)
- Prompt Engineering
- Citation-Based Answering
- Hallucination Reduction Techniques
- Information Extraction
- Document Intelligence Workflows
